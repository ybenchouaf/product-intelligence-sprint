# Agents

This repository is a documentation-first **Product Intelligence Sprint** system.

Read [docs/ai-collaboration.md](docs/ai-collaboration.md) before making changes.

## Quick rules

- Maintain: organization, navigation, links, templates, diagrams
- Prototypes: only after sufficient research exists
- Never: rewrite research for style, remove uncertainty, or drop attribution / evidence / reasoning
- Every commit should improve long-term usefulness
