# Synthesis

Cross-company themes and patterns that outlive any single sprint.

## When to add a note here

- The same structural pattern appears in two or more company briefs
- A market or category insight is not owned by one company
- Methodology or workflow learning should inform future sprints

## Convention

Name files by theme, not company: e.g. `category-name-pattern.md`, `workflow-friction-in-vertical-x.md`.

Each note should:

- Link to contributing company briefs and evidence IDs
- State confidence and open questions
- Avoid duplicating long passages from company folders — summarize and link

Company-specific deep dives remain in [companies/](../../companies/).
