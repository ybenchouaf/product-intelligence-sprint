# Current product — Bridgit

Source: v0.1 seed, sections 1, 7, 8. Facts and inferences labeled.

## Public positioning

**FACT:** Homepage frames product as "AI workforce planning for construction." Navigation groups: planning, resumes, forecasting, AI, integrations, communication.

**FACT:** Moved from narrower "Bridgit Bench" into broader workforce intelligence platform.

**INFERENCE:** Current direction = contractor workforce data system of record + forecasting + AI-assisted planning + enterprise integrations.

## Product modules (public)

| Module | Public capability | Strategic role |
|--------|-------------------|----------------|
| Project Planning | Project hub, maps, roles, assignments, time off, mobile, Smart Suggestions | Day-to-day staffing workspace |
| Forecasting | Supply/demand, roles for hire, utilization, bench cost, what-if planning | Executive and HR planning layer |
| Internal Resumes / Experience Tracking | Digital resumes, project experience capture | Differentiates beyond availability scheduling |
| Bridgit AI | Ask Bridgit, Import Roles, Smart Suggestions, messy-data intake | Current strategic narrative |
| Integrations | CRM, HRIS, ERP, PM, data warehouse connectors | Enterprise adoption and data moat |
| Communication | Alerts, broadcasts, notes, assignment context | Reduces email/spreadsheet coordination |
| Security / enterprise | SOC 2 Type 2, SSO, SAML, permissions, REST API, AWS hosting | Table-stakes enterprise readiness |

## Likely core data model (inference)

Public architecture is limited. Seed infers core objects:

People, Projects, Pursuits/opportunities, Roles, Assignments, Availability, Time off, Skills/experience, Project history, Location/commute, Hiring gaps, Reports/saved views, Notes/communications.

**Confidence:** Medium — inferred from product pages, not confirmed API/schema.

## Integrations named in seed

Autodesk Build, Procore, CMiC, Salesforce, Microsoft Dynamics, Unanet CRM, HubSpot, Workday, BambooHR, UKG, ADP, Snowflake, and others.

## Strategic vs table stakes

**Strategically important:** AI on workforce data; explainable recommendations; cross-system integrations; Experience Tracking / Internal Resumes; forecasting and roles-for-hire; specialty contractor labor forecasting; enterprise/global expansion.

**Table stakes:** SOC 2, SSO/SAML, permissions, REST API, mobile, alerts, dashboards, CSV import/export.

## Product evolution (four phases — inference)

1. Construction software entry — field/PM roots (**FACT:** pre-2019)
2. Workforce planning wedge — Bridgit Bench (**FACT:** 2019 pivot)
3. Enterprise workforce platform — integrations, forecasting, SOC2, APIs (**FACT:** expansion post-pivot)
4. AI workforce intelligence — Ask Bridgit, Import Roles, Smart Suggestions (**FACT:** current)

**INFERENCE:** "Bridgit Bench" still in older materials; current messaging leads with "Bridgit" and "AI workforce planning" — possible platform brand broadening. **Confidence: Medium.**
